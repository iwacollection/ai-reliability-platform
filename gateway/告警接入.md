设计Skills Layer（技能层）。

为什么？

因为目前 Collector 能收集数据，但不会"做事"。

例如：

DescribePod
GetPodLogs
QueryCPU
RestartDeployment
RollbackDeployment

这些其实都是可复用的能力。

在企业里，这些不会散落在 Agent 代码里，而会统一封装成 Skill。

以后：

MCP 暴露 Skill 给 Agent；
LangGraph 调用 Skill；
Multi-Agent 共享同一套 Skill；
Harness 和 Sandbox 也可以复用 Skill。

Skill 会成为整个 AI Reliability Platform 最核心的能力层。

这也是为什么很多公司（包括 OpenAI 的 Agents SDK、Anthropic、Microsoft 等）都越来越强调 Tool/Skill，而不是把所有逻辑都写进 Prompt 里。
