技术选型

我建议第一版技术栈如下：

模块	技术	为什么
Web框架	FastAPI	性能高，自动生成 API 文档
数据模型	Pydantic	类型安全，方便校验
配置	pydantic-settings	多环境配置
日志	structlog 或 loguru	结构化日志
测试	pytest	Python 标准
包管理	uv 或 poetry	管理依赖
容器	Docker	后续部署到 Kubernetes
